VisionBoard AI REST API

Full REST API endpoints for image processing, AI captioning, OCR, object detection, and vector similarity search.

v1.0 OpenAPI Spec
POST /api/upload

Upload an image file (`multipart/form-data`) to execute the full AI Pipeline (YOLOv8, BLIP, EasyOCR, K-Means & CLIP embeddings).

Request Body:
image: [binary image file (.jpg, .png, .webp)]
Response JSON:
{
  "status": "success",
  "image_id": 1,
  "data": {
    "caption": "A young man working on a laptop in an office.",
    "objects": [{"object_name": "Laptop", "confidence": 0.98, ...}],
    "quality_score": 88.5
  }
}
GET /api/search

Natural language semantic search using CLIP 512-d text vector embeddings and Cosine Similarity.

Query Parameters:
?q=Laptop on office desk
Response JSON:
{
  "query": "Laptop on office desk",
  "count": 3,
  "results": [
    { "similarity_score": 92.4, "image": { ... } }
  ]
}
GET /api/object/<image_id>

Retrieve YOLOv8 detected objects and normalized bounding box coordinates.

{
  "image_id": 1,
  "detected_objects": [
    {"object_name": "Laptop", "confidence": 0.98, "box": {"x": 0.1, "y": 0.2, "width": 0.5, "height": 0.4}}
  ]
}
GET /api/caption/<image_id>

Retrieve BLIP AI generated natural language descriptive caption.

{
  "image_id": 1,
  "caption": "A young man working on a laptop in an office.",
  "confidence": 0.96
}
GET /api/ocr/<image_id>

Retrieve EasyOCR extracted text strings embedded in the image.

{
  "image_id": 1,
  "extracted_text": [{"text": "VISIONBOARD", "confidence": 0.98}]
}
GET / DOWNLOAD /api/reports/<image_id>

Generates and returns the downloadable PDF AI Intelligence Report for the given image.

Returns: application/pdf binary download
